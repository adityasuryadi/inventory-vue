## Run Locally

Clone the project

```bash
  git clone https://github.com/adityasuryadi/inventory-vue.git
```

Go to the project directory

```bash
  cd project-name
```

copy config

```bash
  cp .env.example .env
```



Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Akses App

```bash
http://localhost:3000/lists
```
